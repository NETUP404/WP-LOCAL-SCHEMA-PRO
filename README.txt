Potente plugin para SEO local: schemas de negocio, tienda, organización, persona, directorios, ratings, reviews y utilidades SEO local PRO. ¡Totalmente internacionalizable!

== Description ==

WP Local Schema PRO es el plugin más completo para el SEO local de tu negocio:
- **Schema LocalBusiness, Store, Organization y Person** (JSON-LD, compatible Google).
- **Tienda avanzada** (interna o externa), ratings, reseñas, productos, métodos de pago/envío.
- **Enlazador a directorios locales** (Google Maps, Bing, Yelp, TripAdvisor, Páginas Amarillas...).
- **NAP checker** (Name, Address, Phone).
- **Checklist SEO local**, generador QR, validadores rich snippets.
- **Ayuda contextual**, avisos si tienes Yoast/RankMath para evitar duplicados de schema.
- **Panel moderno, accesible y multilingüe**.

**Incluye traducciones en español, inglés, francés, alemán y ruso.**

== Installation ==

1. Sube la carpeta `wp-local-schema-pro` al directorio `/wp-content/plugins/`.
2. Activa el plugin desde el menú "Plugins" de WordPress.
3. Ve al menú "WP Local Schema PRO" para configurar todos tus datos.
4. Personaliza las secciones y activa sólo lo que necesitas.
5. ¡Listo! Tu schema se insertará automáticamente en tu web.

== Frequently Asked Questions ==

= ¿Puedo usarlo junto a Yoast, RankMath u otros plugins de SEO? =
Sí, pero desactiva los schemas de Organization/Person en esos plugins para no duplicar datos.

= ¿Soporta multilingüe? =
¡Sí! Incluye archivos `.po` listos para traducir y es 100% compatible con WPML, Polylang, Loco Translate, etc.

= ¿Es compatible con tiendas externas (ej. Shopify)? =
Sí, puedes añadir la URL y datos básicos de tu tienda externa y se generará el schema correcto.

= ¿Cómo migrar mis datos de Yoast o RankMath? =
Incluimos cuadros de ayuda y avisos para copiar los datos y desactivar schemas antiguos.
